## Описание

Консольная программа на C++, решающая квадратные и линейные уравнения.

Поддерживаются случаи:
- два корня
- один корень
- отсутствие решений
- бесконечное количество решений
- проверка неверного ввода

Используется паттерн «Интерпретатор» для проверки корректности чисел.

---

## Формат входных данных

Три коэффициента:

a b c

Уравнение:

ax² + bx + c = 0

---

## Формат выходных данных

- OK — решение существует
- INF — бесконечно много решений
- WRONG — неверный ввод
- NO SOLUTION — решений нет
